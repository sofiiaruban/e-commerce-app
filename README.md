# Getting Started

## Clone the repository
git clone [https://github.com/sofiiaruban/task-manager-app.git](https://github.com/sofiiaruban/e-commerce-app.git)

## Navigate to the project folder
cd e-commerce-app

## Install dependencies
npm install

## Run the app
npm run dev
